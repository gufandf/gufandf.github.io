使用指南

1. 导入学号：在左侧输入框中输入10位学号（每行一个），点击「导入学号」。

2. 获取课表：系统自动从红岩网校/教务在线获取课表。

3. 开始排班：点击「开始排班」，系统根据课表自动分配值班。

4. 查看/调整：右侧表格展示排班结果，可拖拽学生标签调整班次。

5. 课表管理：点击学生姓名查看课表，可忽略特定课程后重新排班。

6. 导出：支持导出值班表/空课表为 Excel（需联网加载XLSX库），导出/导入排班数据为 JSON。

提示：含"实训""实习""课程设计"等关键词的课程会被自动忽略。
左上角按钮可切换暗色/亮色主题。
