大赛介绍

MCC Game Jam(Minecraft Creativity Game Jam)是由 孤帆Gufandf 和 甜橙猫酱Cat_Sauce 举办的我的世界小游戏地图制作大赛，旨在促进我的世界小游戏地图社区的发展和制作者的交流。参赛选手需要围绕主题，在 72 小时内做出可以游玩的小游戏地图。

比赛规则

请以 参赛手册 为准

  • 围绕本届主题进行游戏制作。
  • 使用最新的Java正式版(1.20.2)制作游戏。
  • 每小队人数不得超过3人。
  • 地图中需要包含制作小组的信息。
  • 比赛过程中允许场外求助和远程协助。
  • 禁止抄袭、偷盗、毁坏他人劳动成果。
  • 地图中禁止包含违反中华人民共和国法律的内容。
  • 如无违反规则的情况，地图版权归参赛者所有。
