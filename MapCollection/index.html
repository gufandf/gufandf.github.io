地图征集计划

介绍

先看一段演示视频

由于学业问题，作者没有时间为此小游戏制作多余的地图。现面向所有热爱Minecraft并有建筑基础的人进行地图征集。作者将把征集到的地图用于小游戏中。

要求

  • 地图大小 48 x 48 x 48
  • 游戏版本 Java 1.18.2
  • 禁止包含引战、黄色、暴力、政治等内容

打包方式

方式一

  1. 到 版本根目录/saves 文件夹
  2. 将存档压缩为 zip 文件

方式二

  1. 使用 结构方块(structure_block) 保存建筑
  2. 到 版本根目录\saves\存档名\generated\minecraft\structures 下找到保存的 .nbt 文件

投稿方式

  • 直接将文件发送到 gufandf@qq.com

注意

  • 发送文件时请加上您的 个人信息! 以便作者对地图进行署名，否则将遵循 WTFPL 协议使用你的地图!

  • 作者将对地图进行一些改动，以适应小游戏的需要
