地图信息

  • 地图版本 Classic 1.3.1(2022/10/27更新)
  • 游戏版本 Java 1.18.2
  • 地图时长 50-80min
  • 建议游玩人数 1-4人

游玩须知

  • 需要 Optfine
  • 开启 移动光源
  • 粒子效果 全部
  • 资源包与解密部分 无关
  • 如是服务端需要将 server.properties 中的 enable-command-block 改为 true
  • 最重要！多读书！
